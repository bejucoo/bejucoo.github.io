DATARAPPER

Datarapper es un proyecto que busca dar una mirada al rap colombiano a través de herramientas y principios computacionales.

Como primer ejercicio se crearon 8 piezas que recomponen/remixean/fusilan carteles de Cine B y de películas R-rated con composiciones gráficas que fueron resultado de utilizar las letras del álbum Hallowinners II de Gordo Sarkasmus como un dataset.

Cada cartel se construyó utilizando la letra de cada canción siguiendo el siguiente proceso:

1. Preparar el dataset: Luego de transcribir las letras, se utilizó Knime para eliminar la puntuación, filtrar las palabras de modo que solo quedaran sustantivos, adjetivos y verbos y crear un archivo JSON con la bolsa de palabras resultante.

2. Obtener las imágenes: Usando la API de Flickr, se realizó una busqueda de imágenes basada en las palabras del archivo JSON creado en el paso anterior.

3. Dibujar las imágenes: De las imágenes obtenidas por cada palabra, se seleccionó una de forma aleatoria y mediante la función blend() de la librería p5.js, se creó un canvas en el cual se sobrepuso cada una de las imágenes selecciondas.

4. Aumentar el canvas: Luego de descargar el canvas de p5.js, este mismo se amplió utilizando Bigjpg (CNN) para poder crear piezas que en un futuro pudieran ser impresas en gran formato.

5. Crear los carteles: Se crearon 8 piezas que recomponen/remixean/fusilan carteles de Cine B y de películas R-rated agregando el título de la canción y otros elementos gráficos.
